# 🌍 Abhaya – Disaster Relief & NGO Management Platform

A full-stack web application designed to connect NGOs, volunteers, donors, and disaster-affected communities through a unified platform for disaster relief management.

---

## 📌 Features

* NGO Registration & Management
* Disaster Relief Campaigns
* Volunteer Registration
* Community Alerts
* Help Requests
* Donation Tracking
* Disaster Reporting
* Organization Dashboard
* Contact & Support
* REST API Backend

---

## 🛠 Tech Stack

### Frontend

* React
* TypeScript
* Vite
* Tailwind CSS
* React Query
* Wouter

### Backend

* Express.js
* Node.js
* TypeScript
* Drizzle ORM

### Database

* PostgreSQL

### Deployment

* Render
* GitHub
